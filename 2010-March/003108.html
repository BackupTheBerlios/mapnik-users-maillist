Hi Carlos,

Welcome to Mapnik! Great thread.

'Better Print Support' would be a great project. One approach would be
to implement svg_renderer (using existing agg/cairo as a basis) . This
way it'll be possible to have full control how features are
rendered/layered. Performance is another area, where we can improve
(cairo is slow). How about fast boost:karma [1] based SVG renderer ?

[1] http://boost-spirit.com/home/2010/03/09/integer-to-string-conversion-karma-fastest-again/

Regards,
Artem
