boost libraries are not configured with threading support ( 
BOOST_HAS_THREADS is undefined ). The easiest way to solve this is to 
download boost source and build fresh libs (see get started on 
boost.org). Mapnik requires boost libs configured and built with  
<threading>multi.BTW, what gcc version  have you got on slackware?
