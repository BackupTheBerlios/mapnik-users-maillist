Aha.. looks like scons feature:). You have to specify all options when 
installing e.g.
"sudo python scons/scons.py install" will ignore plug-ins etc. Try 
something like:

sudo python scons/scons.py INPUT_PLUGINS=all install

Cheers,
Artem.
