Hi Francesco,

i don't think this is possible. You could however translate the contents 
of your way-query as part of your select statement using something like 
that (pseudocode):

(SELECT ST_Translate(way, x, 0) AS way, name  FROM planet_osm_point) as test

On a osm2pgsql database you would specify x in meters.
