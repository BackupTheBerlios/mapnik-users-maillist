There is no need to run server at localhost.  Have a try with MapTiler (www.maptiler.org).  It splits an existing big image to tiles and in addition to that, it creates googlemaps.html and openlayers.html pages which will show the splitted image in web browser.  Look at the code generated by the application and modify as you need.
