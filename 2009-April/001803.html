El Sábado, 18 de Abril de 2009, Erik Johansson escribió:
> What would be the best way of getting a highway to render differntly
> in e.g. Spain and France. Having rendering rules for each highway in
> the DB seems to be easiest, but can you can you use postgis magic or
> mapnik stylesheets todo this automatically?.

We in osm-es are putting together a proposal for using ref:color and 
ref:color_tx (text) and ref:color_bg (background). See 
http://wiki.openstreetmap.org/wiki/Normalización_física#Ejemplos

In order to do this before we get this up and running, you'd need to split the 
planet into countries (osmosis would help, I'm assuming OSM data), add more 
stuff to all highways depending on the country, then insert the data into 
postgis.

I don't know if you could do that directly into postgis - e.g. select a bunch 
of linestrings inside a given polygon and update their records.

And anyway, you'd need to do some mapnik stylesheet magic to get the shields 
right.
