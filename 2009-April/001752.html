Hi Christian,
Most likely you need to install vc90 runtime libraries on your xp. You
should be able to find all required files and install instructions on MSDN
site.

Here are steps I suggest you follow  :

1. Download "Dependency walker"
2. Open mapnik.dll in DW
3. If you see runtime DLL ( I think it's called msvcrt90.dll) is missing,
download it from MSDN  and install.

HTH
