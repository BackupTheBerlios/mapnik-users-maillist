'buffer_size' is a property of the Map object and defaults to 0 - good  
result have been had but setting that value to at least half the size  
of your tile, so often m.buffer_size = 128

I've added mention of it to the XML reference page:

http://trac.mapnik.org/wiki/XMLConfigReference
