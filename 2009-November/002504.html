Indeed you can, although I've not been able to set up correctly yet,  
due to lack of time and attention (hopefully only that!).

The idea would be that you read in the WMS feed as as bitmap using the  
GDAL WMS minidriver:

http://www.gdal.org/frmt_wms.html

I got as far as the image showing up, but slightly shifted to the  
north, so I may just need to tweak the params. I've only worked on  
this programmatically when hastily trying to add support for WMS  
layers within Quantumnik (http://bitbucket.org/springmeyer/quantumnik/src/tip/sync.py#cl-553 
), so trying to set up one layer might be a bit more feasible.

Let us know how it goes.
