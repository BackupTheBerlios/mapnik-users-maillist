Hi,

How do I request all layers within a mapfile (osm.xml) for a wms request
against a Mapnik OGC WMS Server?

The hard way is to have all 45 layers as a comma separated value assigned to
the layers parameter.

The easy way (like it is done in UMN MapServer) is to have a layer parameter
at the top (header) part of the mapfile that then can be referenced within
the WMS request.
