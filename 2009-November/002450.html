Hi,
 
WMS would maintain the quality only if BBOX and WIDTH and HEIGHT parameters of the request happen to suit exactly the dimensions of the prerendered tiles.  Otherwise tiles need to be resampled before generating WMS output.  Loss in quality will be worse if tiles are also re-projected to some other projection.
