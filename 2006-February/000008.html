A the moment following vector geometries can be visualized:

    * Point
    * LineString
    * Polygon (with holes)
    * Multipoint
    * MultiLineString
    * MultiPolygon

Basically all geometries as per OGC Simple Features Specification rev1.1 
except GeometryCollection (which I'm planning to add for compliance).
