Presumably you ran:

python scons/scons.py install ?

Buy default, everything gets installed under /usr/local ...

If you did that, then /usr/local/lib probably isn't in your ld.so path.  
you can look at /etc/ld.so.conf for details on that.

Also note that if you want the CVS version, you can stop python building 
by using:

python scons/scons.py BINDINGS=none

J.F.
