Looks good!

To save time and build only 'release multi-threaded' libs you could add

"-sBUILD 
<http://www.boost.org/tools/build/v1/build_system.htm#build>=release 
<threading>multi"

Also you specify only required libraries e.g  --with-thread 
--with-python  --with-regex

You might have to:

export PYTHON_ROOT=/opt/path_to_python PYTHON_VERSION="2.xx"
 
if you're using non-standard python installation
