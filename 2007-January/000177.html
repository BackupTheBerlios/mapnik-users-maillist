Nick,

 Take a look at TileCache : http://labs.metacarta.com/wms-c/ . This, I
believe, is what OSM is using and it works well with Mapnik.

 It's written in python but doesn't require you know python to use it;
its all configured in a single text file and runs as a cgi or an
apache module. In order to pregenerate tiles, you can use the
Client.py program:

  python Client.py "http://localhos/tilecache/tilecache.py?" layer 0 5

(Where 0 - 5 are the zoom levels.)

Hope this helps and happy new year!,

matt
