Also, you likely don't want to put the paths for both python versions. If you are using Python 2.6, just use the path to that.

To debug things further, do:

$ python
import sys
print sys.path

That will show all the paths that are searched, and you need to make sure your edits to the PYTHONPATH are working such that the new paths show up in sys.path
