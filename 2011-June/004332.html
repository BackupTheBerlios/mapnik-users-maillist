And here's the performance of the OSM tile server:

http://munin.openstreetmap.org/openstreetmap/yevaud.openstreetmap/renderd_processed.html

If you look at the yearly graph, you can see the difference that came 
with the tile server upgrade (from a fast SATA array to SSD). Scale is 
in metatiles (= 64 tiles).

The OSM tile server uses a full planet extract without simplified 
geometries (standard osm2pgsql setup) and also processes updates along 
the way. Hardware details are available here: 
http://wiki.openstreetmap.org/wiki/Servers/yevaud
