Hi all,

Nik2img 0.2.0 adds a bunch of new features and can now be installed  
with a simple:

$ python setup.py install

Nik2img.py can now directly read Cascadenik MML (Mapnik Markup  
Language), be imported as a python module, write worldfiles to  
georeference output, and stream the output image as a string.

For a full list of additions see the release changelog: http://mapnik-utils.googlecode.com/svn/tags/nik2img/nik2img-0_2_0/changelog.txt

The script can be downloaded from: http://code.google.com/p/mapnik-utils/

Dane
