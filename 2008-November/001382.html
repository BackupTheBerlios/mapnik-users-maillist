I did not need to specify BOOST_VERSION=1_37.

cf:
https://trac.mapnik.org/ticket/90

my build comands now look like:

# to clean
/usr/bin/python scons/scons.py
BOOST_INCLUDES=$HOME/opt/boost/include/boost-1_37
BOOST_LIBS=$HOME/opt/boost/lib BOOST_TOOLKIT=gcc41 -c

# to complie
/usr/bin/python scons/scons.py
BOOST_INCLUDES=$HOME/opt/boost/include/boost-1_37
BOOST_LIBS=$HOME/opt/boost/lib BOOST_TOOLKIT=gcc41

# to install
sudo /usr/bin/python scons/scons.py
BOOST_INCLUDES=$HOME/opt/boost/include/boost-1_37
BOOST_LIBS=$HOME/opt/boost/lib BOOST_TOOLKIT=gcc41 install
