Arnauld,

If you have rendered tiles to the filesystem already, then you likely only need set up your mapping client to view them.

If you are using OpenLayers, this would mean using one of the layers like OpenLayers.Layer.OSM or OpenLayers.Layer,TMS (depending on the tile storage scheme).

If using Google Maps, then see GTileOverlay.

You say below that you have used Mapnik to generate tiles, but that you rendered them with Geotools. What is GeoTools?
