Hi Adam,

On Mon, 14 Sep 2009, adam at spatialsystems.org wrote:
> I need to make a desktop GUI that renders a ShapeFile map with thematic
> mapped data points where you can pan and zoom.  All I know is Perl
> without any GUI background.
>
> What language and GUI toolkit is recommended to use with Mapnik to make
> something like this?  Is there a tutorial that covers this?  I haven't
> been able to find one anywhere.

You might find the combination of the Desktop GIS QGIS (Quantum GIS) and a 
plugin for it, "Quantumnik" interesting:

 	http://bitbucket.org/springmeyer/quantumnik/wiki/Home

As far as I know, the Plugin allows to use Mapnik as a rendering engine 
inside QGIS (with its zomming and panning features). It has some more 
capabilities, of course, but these seem to be what you are looking for.

QGIS is scriptable e.g. using python, if you need it.

Yours, Holger
