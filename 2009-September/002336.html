Hello,

I had the same problem with gdal plugin.

Actually I had one DLL (GDAL15.DLL) missing in the execution environment.
You might have the same problem with OGR, so try to run DLL dependency
walker tool (http://www.dependencywalker.com) against the ogr.input (I did
it with gdal.input, renaming it in gdal.dll) and found the DLL I was
missing.

This missing DLL had to be put in the execution directory (not the same as
plugin directory most of the time)

Regards.

Nicolas
