On Oct 27, 2008, at 8:30 AM, Jamie Robertson wrote:

> ...
>
> what is the difference between setting the Map srs and the Layer  
> srs?  they should be the same, correct?
>
> ...
>


Not necessarily, in fact not usually. The Layer srs specifies the  
input projection of the source layer, the Map srs specifies the output  
projection of the result map. It's common for layers to be in latlong  
and maps to be in mercator, for example.
