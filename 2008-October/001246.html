First you should make sure the font that you are using has the required
characters. Mapnik supplies an old version of the DejaVu fonts and they
do not good have international coverage. 

Mapnik can use other fonts but the instructions on how to change them
depend on what is calling Mapnik. What tools are you using to call
Mapnik, one or the python scripts or something else?
