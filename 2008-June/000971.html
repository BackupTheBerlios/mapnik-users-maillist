Hello,   I'm trying to render a raster file (using gdal), but the image
file doesn't necessarily have default RGB bands set.    It appears that
the mapnik gdal plugin gets those RGB bands from the file and if they
don't exist it treats is as grayscale.

Anyone encountered this at all?  I'd like to specifically give it the
RGB bands.  This is especially important for hyperspectral imagery where
a user may want to specify different viewing bands every time.    It's
not practical to have to create a different copy of the file for every
possible combination of bands in even a simple 7 band multispectral
image much less one with 128 bands!
