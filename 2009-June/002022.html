Hello Adnan,

Note that Mapnik does not support re-projection of rasters. So you  
have to make sure to:

1) reproject/warp your TIFF to the OSGB srs first (I use 'gdalwarp'  
for this)
2) remember to set the layer.srs parameter to the OSGB proj4 string

I noticed that you are not doing this in your attached script, so  
you'd need something like:

lyr = Layer('world')
lyr.datasource = Gdal(file='c:\\temp\\london-20k9.tif')
lyr.srs = osgb

or

lyr.srs = m.srs

Dane
