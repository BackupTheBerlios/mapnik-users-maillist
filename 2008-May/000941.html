with an offset label using "shield" placement type, so the point is part 
of the label and not a "symbol"? This would be useful for labeling 
cities with both a marker (star, circle, etc) and label at smaller 
scales where not all the cities will be labeled due to placement 
conflicts (especially useful in dense city-regions with many good-sized 
cities close to one another).
