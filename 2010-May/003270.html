Latest nik2img adds geotiff output (georeferenced tiff) to the list of one-liner output formats.

Basically, if the GDAL utilities are installed you can now do:

$ nik2img map.xml image.tif

Internally nik2img builds up a world file and dispatches to gdal_translate to create the tiff and give it the correct spatial reference.

You can also now pass an optional flag to zip up the output:

$ nik2img map.xml image.tif --zip
