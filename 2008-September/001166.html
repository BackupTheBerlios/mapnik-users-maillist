I posted some more-streamlined OS X installation directions on the Mapnik
Trac wiki:
http://trac.mapnik.org/wiki/MacInstallation

Unlike the previous guide (http://trac.mapnik.org/wiki/MacInstallationSource),
this guide depends on MacPorts to grab the dependencies.

In theory, this guide should work pretty well from a vanilla OS X Leopard
install and has been somewhat tested, but in practice life is complicated :)

Let me know what needs patching or edit the wiki yourself! (In particular,
the patch provided assumes that you have installed PostgreSQL. Ignore those
lines if that isn't the case.)
