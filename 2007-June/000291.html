> Hello,
>
> I noticed that some zoom levels doesn't render the sea correct. In  
> most
> zoom levels the sea is blue, but in some cases the sea is rendered  
> grey
> and there's only a blue line at the coast line. Do you know this
> problem? If not I'll take a screenshot.
>
> Any ideas why this happens and how to solve?
>


Yes, this is a data issue. Boundaries  are derived from global  
shoreline dataset which requires some preprocessing to create  
polygons. I've converted most of the UK, but we need more then one  
person for the whole world. Where there are no polygons you get grey  
background + blue line .

At the moment I'm waiting for OSM coastline to be fixed in db, then  
it should be possible derive better polygons.

BTW, osm-devel is probably better place to discuss this.
