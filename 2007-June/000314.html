Hi Luiz,

On 28 Jun 2007, at 16:07, Luiz Vital wrote:

> Hi all,
>
> How can I make a Rule filter with an boolean field in a postgis  
> datastore?
>
> Haven't get any clue by testing and looking at the source.
>
> Some examples that failed:
>
> rule.filter = Filter("[PUBLISHED] = TRUE")
> rule.filter = Filter("[PUBLISHED] = 'TRUE'")
> rule.filter = Filter("[PUBLISHED]")
>
> Is this implemented somehow in the Filter parser?
>

At the moment attributes values can be one of : int, double, wstring.  
There is no explicit support for boolean type.
I think we should add support for boolean types , would you mind  
submitting a ticket, please?

As a workaround you could try filter expression like PUBLISHED] = 1  
(untested). Let me know.
