> Hi Nuno,

> As a starting point you could replace Shapefile with PostGIS in

>

>
> .........
> lyr.datasource = PostGIS (host='localhost', user='xxx', pass='xxx',
> dbname='your_database_name',
> table='your_table_name_or_select_statement')

>
> Obviously you need to load data into postgresql first. There is handy
> utility shp2pgsql (part of postgis dist).
> You don't have to provide all parameters as there are reasonable defaults.
> If you're working with large data table consider passing
>
> estimate_exetent='true' when creating datasource object e.g.
>
> lyr.datasource = PostGIS (..., ..., estimate_extent='true')
>
> and index geometry column(s).
