Solved,
Was a problem in Postgresql includes and libs.
For those who uses Ubuntu this solves:

python scons/scons.py PGSQL_INCLUDES=/usr/include/postgresql
PGSQL_LIBS=/usr/lib/postgresql/8.1/lib install
