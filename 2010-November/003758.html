On Tue, Nov 23, 2010 at 7:36 PM, Dane Springmeyer <dane at dbsgeo.com> wrote:

> Hi John,
>
> The Mapnik bindings support python 2.5 --> 3.x (in trunk). However to
> support a given major version the bindings must be built against the headers
> of a major python version.
>
> Only bindings built against python 2.5 and 2.6 are currently available for
> the windows binaries (and the mac binaries).
>
> But, you could create a ticket at trac.mapnik.org requesting 2.7 support
> so that we consider it before the next release.
>
> Dane
