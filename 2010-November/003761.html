Mapnik windows binaries are 32bit, but work fine on 64 bit windows 7  
(at least I tested them this way within vitualbox)

As far as Mapnik running within apache/mod_wsgi my assumption is that  
you must match arch exactly. So even if your os is 64 bit capable both  
Mapnik and apache must be the same arch.

So, I would guess 32bit Mapnik would work with 32 apache on 64 bit  
windows server if apache itself works there. But, let us know.

64 bit Mapnik binaries are a desire, but I would not hold your breath  
for them because it's an unknown of whether we can compile each mapnik  
dependency 64 bit on windows to be able to ship a 64bit Mapnik version  
for windows.

Dane
