Am 15.12.2009 20:39, schrieb Aryeh Leib Taurog:
> Does the mapnik API provide an easy way to get the maximum
> PointSymbolizer icon size for a given zoom, so I can determine myself
> whether or not the icon will run off the edge onto the next tile?  Is
> there an easier way to do this?

Usually, this problem is circumvented by always rendering with a larger 
canvas with an additional border around the tiles, which is discarded. 
This way, icons and texts overlapping an edge are drawn twice and part 
of them is on one tile and part on the other.

Mapnik has a feature to create such a buffer border.
