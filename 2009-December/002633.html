compile and install boost with the default settings and it should  
install in /usr/local, then do:

python scons/scons.py BOOST_LIBS=/usr/local/lib BOOST_INCLUDES=/usr/ 
local/lnclude

If that does not work then send the terminal output and attach the  
'config.log' file.
