I've released the first public offering of the Mapnik OS X Framework  
installers:

http://mapnik.org/news/2009/dec/16/osx_installers/

They currently provide the latest from the 0.7 branch which will be  
soon promoted to the next release.

My hope is that one side benefit of having development releases as  
easy-to-install binaries is that more users will help find problems  
and create tickets before each release.

If you have suggestions for improving the installers please file Trac  
tickets against the "Mac OS X Binaries" component.

Dane
