Hi Tom,

Your database server is not configured to allow outside connections. To
enable, you'll need to modify pg_hba.conf and restart db server. Please,
follow postgresql documentation :

http://www.postgresql.org/docs/8.4/interactive/auth-pg-hba-conf.html
