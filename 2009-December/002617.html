Nop,

install cairo's python bindings like:

sudo apt-get install python-cairo

and I think the problem will be fixed.

If that works then its this problem: http://trac.mapnik.org/ticket/392  
which I fixed a while ago and was fixed in the debian packages, but it  
must have slipped through the cracks and missed being included in the  
ubuntu packages.

The workaround is to install python-cairo. This workaround is not  
needed on any Karmic systems with the desktop stuff because python- 
cairo comes pre-installed.
