> situation is short, single segment roads, that need to be labelled.

You could take each of those short segments and create a copy of it in the
database where you make the segment longer. Then draw the original segment
as road and use the longer segment to draw the label with.

I have not tried this, but it should be reasonably easy to do with a bit
of math. Can probably be all done in the database. If the longer segment
still has the same center as the smaller one, labels should look alright
is most cases. Chances are you'll get overlapping labels that way, though.
