On Dec 31, 2007 11:03 PM, Steve Smith <ssteve52 at yahoo.com> wrote:
> I have more than one my system. I would like to point to the libz in
> /usr/lib64 instead of the default path of /usr/local/lib. I have attempted
> to change the path when running scons, but with out success. I tried using
> Z_LIB=/usr/lib64 and ZLIB_LIB=/usr/lib64, but scons is still attempting to
> use the one in /usr/local/lib. Can the libz path be changed?

There is no specific path for a library. There is a library path for
*all* libraries. So if you don't want something in /usr/local/lib you
need to remove it from your search path entirely.

Perhaps a better solution is to simply remove the libz from /usr/local/lib?
