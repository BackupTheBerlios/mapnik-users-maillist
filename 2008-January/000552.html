Hi,
we are planning to give the XML format a little revision. Some changes  
were discussed on mapnik-dev back in October. Here is a list:

http://trac.mapnik.org/query?keywords=XML

Because file format changes should not happen to often we would like  
to collect all your ideas and criticism concerning the file format.  
Please paste your XML snippets (if you have any) on this wiki page

  http://trac.mapnik.org/wiki/XmlFormatDiscussion

and discuss them here on the list. You need a Trac account for that.

The format change is scheduled for v0.6, but to help with migration I  
will add a load_map_v2() asap.
