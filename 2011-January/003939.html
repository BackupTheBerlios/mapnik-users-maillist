Hello Peter,

There is also Quantumnik. It is a Quantum GIS plugin which basically
allows you to:
- render a Mapnik mapfile,
- export a Mapnik mapfile from the current map definition.

Infos at: https://bitbucket.org/springmeyer/quantumnik/wiki/Home
QGIS plugin repository: http://qgis.dbsgeo.com/
