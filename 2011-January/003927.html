Hi Aníbal,

The request is asking for WMS VERSION=1.3.0, so you/mapfish need to specify exceptions in that format:

WMS 1.1.1
EXCEPTIONS=application%2Fvnd.ogc.se_inimage

WMS 1.3.0
EXCEPTIONS=INIMAGE

Other valid values in WMS 1.3.0 are XML and BLANK.

See you!
