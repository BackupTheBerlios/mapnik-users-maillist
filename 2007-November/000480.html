On 7 Nov 2007, at 17:16, Tyler Durden wrote:

> Hi,
> I'm trying to draw a polygon from a structure(or a list, KML or
> whatever etc) of points.
> There is a way to define a Datasource besides XML, Shapes or Postgis
> with python bindings?


No, not at the moment. I'm planning to add support for creating  
Features/Geometries from Python. I was thinking about using OGC WKT:

geom = from_wkt('POINT(1233 1321)')
geom2 = from_wkt('LINESTRING(13213 43214,54325 65436, 65436 65437)')

and/or direct interface, somethings like :

pt = Point(-1123,123)
line = LineString()
line.addPoint(123,123)
line.addPoint(3324,4324)
