Am Mon, 22 Oct 2007 18:35:33 +0200 schrieb Gilles Bassière:

> Hi List,
> 
> I have a number of issues to design a map:
> 
> 1) Font management
> - How can I add new fonts? Is it possible to use system-wide fonts? I 
> tried to copy .ttf files to lib/mapnik/fonts/ but I can't use it.
> - Is there a way to know the "face_name" that  mapnik expects? For 
> exemple, according to Gnome Font Viewer, the name of DejaVuSans.ttf
> is "DejaVu Sans" but Mapnik will work only with "DejaVu Sans Book",
> which tool should I use to figure this name?

The autoconf/automake installation doesn't install the fonts. I simply
use the system fonts on my Linux system. Like this:


  freetype_engine::register_font("/usr/share/fonts/truetype/ttf-dejavu/DejaVuSans.ttf");
  freetype_engine::register_font("/usr/share/fonts/truetype/ttf-dejavu/DejaVuSans-Bold.ttf");
  freetype_engine::register_font("/usr/share/fonts/truetype/ttf-dejavu/DejaVuSansMono.ttf");

regards
Andreas
