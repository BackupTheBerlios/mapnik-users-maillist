Mapnik 0.4.0 is now available in Debian unstable; if you're running
Debian unstable or testing (with unstable packages available) then you
should be able to install mapnik in seconds. Eg:

 # apt-get install python-mapnik

See
http://packages.debian.org/unstable/source/mapnik
for full details.

Feedback direct to me or the Debian BTS welcomed!
