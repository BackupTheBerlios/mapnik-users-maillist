Hi Ludovic,

The Oracle plugin is not available for the Windows binaries  
unfortunately, so you'd need to compile it yourself.

I've personally not used this plugin so I can't advise about how easy  
it is to compile on any platform, but this wiki page at least has  
links to the Oracle client libraries needed:

http://trac.mapnik.org/wiki/OCCI

Of note: currently the SCons build system used to compile both Mapnik  
core, its python bindings, and the plugins is not supported on  
windows. Therefore, if you are determined to get the plugin built on  
windows one step could be to first use the SCons build system to  
compile it on Linux, then borrow from the g++ command line arguments  
and manually use those to form up a set of compile commands for  
windows in visual studio, or whatever build environment you might use.

Dane
