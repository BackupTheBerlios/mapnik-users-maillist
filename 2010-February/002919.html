hi,
i am having trouble using multiline SQL statements in an XML
Postgis datasource:

This works:

	<Parameter name="table">(
	select id,border,complete from completeborders where completeinborder(border, 62611) and complete is not null
	) as admin </Parameter>

This doesnt:

	<Parameter name="table">(
		select	id,border,complete
		from	completeborders
		where	completeinborder(border, 62611)
		and	complete is not null
	) as admin </Parameter>

whereas the latter is much more readable IMHO.
