2011-04-14 Yann Barraud wrote:
> - I'm working on windows, and when trying to add an osm file in XML, I get
> an error saying I have no OSM plugin installed. Which seems to be the case.
> Where can I get OSM plugin (.bin & .input) ?

  Please note that OSM plugin does not support these:
  1. Relations (so for example admin borders going through rivers,
roads etc. will NOT be rendered, huge forests captured as a number of
non closed vectors in a closed relation will NOT be rendered)
  2. Multipolygons (so you will NOT get islands in lakes, non forested
areas inside forests, gaps in buildings etc.).

  I'm not sure if you can add OSM plugin "on-line/dynamically". I had
to recompile mapnik (which was easy to do on linux using src.rpm, on
win it will probably be more difficult...)
