Yann,
There are a few howto's, but I am not sure if there is one for windows -
see http://wiki.openstreetmap.org/wiki/Mapnik and
http://wiki.openstreetmap.org/wiki/Rendering.

I do not know if the OSM plugin for mapnik works reliably - I have only ever
used mapnik by importing OSM data into a postgresql database using
osm2pgsql.

If I wanted to render a map on a windows machine, I would use a crude
approach of installing virtualbox to run Ubuntu on the windows machine, and
install the rendering tool chain on the Ubuntu installation.  It is not a
fast way of doing it, but at least there is some step-by-step documentation.

Someone else may be able to point you to a Windows tutorial.

Sorry I can't be more help!

Graham.
