Your tools are very similar to mine. You're building on intel mac,  
right?
Could you try first disabling python bindings :

python scons/scons.py <your options here>   BINDINGS=none

(the above should build libmapnik.dylib)


and then re-enabling them again:

python scons/scons.py <your options here>

Let me know if the above solves the problem.
