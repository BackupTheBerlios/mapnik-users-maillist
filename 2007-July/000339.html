On 18 Jul 2007, at 19:33, Jeff Cooper wrote:

> Hello,
>
> I am setting up mapnik to connect to a PostGis database and have a few
> questions about the datasource configuration.
>
> 1) How do I specify a password to the database?  In the examples I  
> have found
> there is no password specified.


<Parameter name="password">[your password]</Parameter>


> 2) The table parameter is a bit cryptic.  What should be specified  
> here?
>
> Here is my current Datasource section:
>
>                 <Datasource>
>                         <Parameter name="type">postgis</Parameter>
>                         <Parameter name="host">{Fully Qualified Host
> name}</Parameter>
>                         <Parameter name="port">5432</Parameter>
>                         <Parameter name="user">{username}</Parameter>
>                         <Parameter name="password">{password}</ 
> Parameter>
>                         <Parameter name="dbname">{database}</ 
> Parameter>


>                         <Parameter name="table">{{What goes
> here?}}</Parameter>
>

You can pass a table name e.g.   roads . Make sure the table is  
properly referenced in 'geometry_columns'
Or you can pass a query :   (select * from roads where  
type='motorway' order by ....) as myroads.
