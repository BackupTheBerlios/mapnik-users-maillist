On Friday 09 February 2007 10:02, þafak çýrað wrote:
> Hi,
> There are two methods setMinZoom,setMaxZoom defined in layer.hpp.
> Can i use them for setting visibility scale of a layer?
> If so,does rendering process consider these values?
Yes, you can. But I suggest using  min/max_scale on Rule object for greater 
flexibility.
 
> Any python code sample illustrating visibility scale will be appreciated.

my_style = Style()
rule_0 = Rule()
rulle_0.min_scale = 1000
rulle_0.max_scale = 10000
# add some symbolizers etc
.....

my_stylerule_1 = Rule()
rulle_1.min_scale = 10000
rules.append(rule_0)
...

Now, for scales between 1000-10000 rule_0 will be applied and rule_1 for  
scales >= 10000.  Actual numbers depend on your projection. And there's no 
active rules for scale < 1000.
