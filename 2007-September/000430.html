Hi Tyler,

Mapnik is using 'standardized rendering pixel size' which is derived  
from  ~ 90 DPI :

( 1/ 90 )  * 0.000254 ~= 0.00028

I think setting DPI = 90 (or even 90.833) in OL should give you right  
numbers.  If you think this should be configurable Mapnik (same as in  
OL), please, sign up to  trac.mapnik.org and create a ticket (This  
would also help testing new Admin plug-in ;) .
