Hi Jill,

The mapnik.ogcserver package is not on your python's path.

Try this from a python interpreter:

import sys
print sys.path

This will tell you where python is looking for packages. You can either modify python's path or make sure mapnik.ogcserver is in one of the directories shown in sys.path.

Hope this helps!
