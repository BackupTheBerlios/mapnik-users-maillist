Hello,

I've added a python demo in the demo/python directory to subversion, as 
well as a very very preliminary try at generating Python API docs using 
epydoc (http://epydoc.sourceforge.net/).  These are located in 
docs/epydocs.  There are very little "docstrings" for now, and the 
Boost.Python layer adds some "garbage" in there, but at least there's a 
reference.

The demo is probably the best place to start right now, it's a fully 
functional script with plenty of comments ... it doubles as a tutorial.

Enjoy!
