Hello,

Try to build the sources from SVN with:

./autogen.sh --prefix=/usr/local && make && make install

use the --help parameter for more help.

I maintain the autotools build environment for mapnik. It's not yet
tested with OS X, but may be interesting to know if it works. If you
get an error send me the log files.
