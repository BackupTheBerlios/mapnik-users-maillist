Regarding building Mapnik on Mac OS:

Its been a busy week on the list. Lots of new users trying to get  
going with mapnik and getting hung up on the install stage. Not a lot  
of fun, but a good motivation for better docs.

Thanks so much to Justin for helping with such keen solutions and for  
posting patches.

Something I'd been meaning to do for a while is post an installation  
guide on Mapnik Trac. Justin's nice code suggestions prompted me to  
clean up and add to my build notes.

So, I've started this page: https://trac.mapnik.org/wiki/ 
MacInstallation which you all should add to and refine or even split up.

For those of you still struggling with Envelope or NoneType problems  
you'll note that I listed your unresolved problems. Even better would  
be for you to file formal tickets on the trac site.

Cheers,

Dane
