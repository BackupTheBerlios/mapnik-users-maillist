Hi, I committed an experimental branch a few weeks ago with a visual 
studio project for building mapnik, you can find it in:

svn://svn.mapnik.org/branches/experimental-pdf

In the msvc directory is a solution which will build mapnik and all the 
other bits. Have a look at the readme.txt file for more info.

The branch was for some experimental pdf stuff, so you'll want to copy 
the msvc directory to the svn trunk you've already checked out and 
remove all the pdf stuff from the projects (not much really).
