Hi, I'm guessing you are using different third party libraries to me 
(freetype/icu4c/libjpeg/libtool/libpng/proj.4/libtiff/zlib). The project 
expects the installation to have been done according to 'readme.txt' in 
the msvc directory, with all the environment variables (LIBJPEGDIR, 
LIBPNGDIR, etc.) set. The readme file lists where I got all the 
libraries from. If you have different libraries installed, then you'll 
need to update each of the projects 'additional include directories' and 
'additional library directories' to point to the new place.


If this doesn't work, send me some of the error messages and I'll see if 
  I recognise the problem.
