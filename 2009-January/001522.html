Mapnik-users,

We are beginning to discuss the next Mapnik release, 0.6.0, on the  
mapnik-devel list and #mapnik irc on freenode. If you are interested  
in participating, feel free to join the discussion.

The 0.6.0 release will include Cairo rendering support, new Unicode  
support with ICU, and a much improved Scons build system, as well as  
numerous other improvements and bugfixes.

However, we've still got a healthy set of tickets against the 0.6.0  
release that we're currently working on: http://tinyurl.com/0-6-milestone-priorities

Here's what you can do:

  - Show your support for Mapnik by signing up at Ohloh.net and  
hitting the 'I Use Mapnik' button: http://www.ohloh.net/p/mapnik
  - If you have any outstanding issue not addressed in the already  
closed tickets(http://tinyurl.com/Mapnik-Closed-Tickets), please post  
them to Trac __now__. We may need to defer them till the next release,  
but it is best to have to them there asap.
  - If you have any opinions/thoughts on the existing tickets against  
the 0.6.0 milestone, please comment on them.

Thanks!

Dane
