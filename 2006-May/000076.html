Samuel,

To create PostGIS Layer for WMS server ( given you set up spatial 
table(s) ):

lyr = Layer(name='test', type='postgis', host='localhost',dbname='gis',
user='postgres' ,table='select * from mastermap',password='postgres')

Please, have a look in ${MAPNIK_SOURCE}/docs/ogcserver/readme.txt
Also there is on-line tutorial  at http://www.perrygeo.net/wordpress/?p=49
for general instructions
