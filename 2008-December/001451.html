Sebastian,

nik2img is meant to generate single images.

For generating a tile set you'll want to use generate_tiles.py:

http://wiki.openstreetmap.org/index.php/Deploying_your_own_Slippy_Map#Creating_tiles_using_Mapnik_and_generate_tiles.py
