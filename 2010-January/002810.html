Hi Artem,

as I promised, here is an installation "How To" for Suse Linux Enterprise 10
Service Pack2, 64 Bit:

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
Get and install Subversion:
download subversion-1.6.6-1.4.x86_64.rpm
also, install libexpat0-1.95.8-65.1.x86_64.rpm (needed 4 neon) and
neon-0.26.1-10.1.x86_64.rpm
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
mkdir src
cd src
svn co http://svn.mapnik.org/trunk/ mapnik
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
install (YAST) libjpeg-devel, libtiff-devel, pkgconfig,
freetype2-devel, xml2-devel, cairo-devel, python-devel,

Try without: libicu-devel,  boost-devel, libpng-devel
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
install libproj0-4.7.0-1.1.x86_64.rpm THAN
libproj-devel-4.7.0-1.1.x86_64.rpm
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
install icu 4_2_x
1. gunzip -d <  icu4c-4_2_1-src.tgz | tar xvf -
2. cd icu/source
3. ./runConfigureICU Linux
4. gmake
5. gmake install
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
install boost:
1. cd opt/
2. svn co http://svn.boost.org/svn/boost/trunk boost
3. cd boost
4. ./bootstrap.sh
5. ./bjam --with-regex --with-iostreams --with-filesystem --with-thread
toolset=gcc -sNO_BZIP2=1 -sHAVE_ICU=1 -sICU_PATH=/usr/local/ -a install
-> make sure all necessary Libs are compiled! If not, run bjam without any
option!
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
install libpng 1.4.0
1. /configure --prefix=/where/ever
2. make check
3. make install
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
install Mapnik
1. cd mapnik
2. python scons/scons.py BOOST_INCLUDES=/usr/local/include/boost/
BOOST_LIBS=/usr/local/lib/ ICU_INCLUDES=/usr/local/include
ICU_LIBS=/usr/local/lib PNG_LIBS=/where/ever
3. export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/opt/libpng/lib
-> scons: done building targets.
4. python scons/scons.py install
-> scons: done building targets.
5. Test Mapnik:
a. python ... import mapnik
b. cd demo/python
c. python rundemo.py
-> Should create 4 demo maps within this folder
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++


Thanks a lot for your help!
