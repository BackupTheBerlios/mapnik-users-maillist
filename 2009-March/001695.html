Hey Max,

Welcome!

First thing I'd try is to confirm your paths a second time by doing:
$ python
 >>> import sys
 >>> print sys.path

And confirm that your lib directory with the DLL truly shows up in the  
sys.path list. I've needed to set the path for both the user and the  
system in the past to get things to work properly.

Next thing I would try is to test with python 25 (if you have it on  
your system). I can't confirm that python 26 works with 0.5.1,  
although it should work fine with the upcoming 0.6.0 release.

Dane
