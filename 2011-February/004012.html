Passwords are not supported when reading through Mapnik's OGR plugin, which is what is happening because QGIS is reading through OGR.

It is better if you add the Postgres layer using the native PostGIS data provider in QGIS, which will then map to Mapnik's native postgis plugin.

So, add your layer by going to Layer -> Add PostGIS Layer rather than Layer -> Add Vector Layer.

Dane
