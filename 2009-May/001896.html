Hi all,

I'd like to announce the public release of spreadnik 0.2.

Spreadnik is a tool to help manage large mapnik XML stylesheets, by allowing a 
cartographer to edit spreadsheets instead of a large text document.

I wanted to code it because I *hate* scrolling up and down whenever I want to 
change some width in some zoom level in the XML stylesheet. The main goal of 
spreadnik is to provide the user with compact stylesheets that can be changed 
and copy-pasted quickly.

For now, spreadnik lives in the openstreetmap SVN repository, along with some 
example spreadsheets:

http://trac.openstreetmap.org/browser/applications/utils/spreadnik

And you'll find some documentation and tutorials in the OSM wiki:

http://wiki.openstreetmap.org/wiki/Spreadnik


By no means I want to replace cascadenik, but I hope spreadnik will be a 
worthy contestant in the battle for understandable stylesheets ;-)



Thanks to everyone that makes mapnik possible, and cheers to all,
