Hey all,

Mapnik 0.6.0 (and Portfile) was just accepted to Macports this  
afternoon (http://trac.macports.org/changeset/50499).

So, as soon as all the mirrors sync Macports users will be able to do:

$ sudo port selfupdate
$ sudo port install boost +pyhon26+icu
$ sudo python_select python26
$ sudo port install py26-mapnik

Cheers,
Dane

* Note: using Mapnik installed via Macports requires that you switch  
to using Python 2.6 installed from Macports
