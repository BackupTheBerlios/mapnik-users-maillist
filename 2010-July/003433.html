> I've also written a MDI based pyqt4 program as a example(the viewer).
> There still may be some bugs. I've only tested it on win32 platform.
> 
> mapnik map viewer features:
> 1. Open and view multiple mapnik maps in one MDI area. Drag to pan and wheel to zoom map.
> 2. A dockable widget shows a tree view of map's attributes of current activated subwindow in the MDI area. The value of blue name can be modified. For boolean values, '1'/'ture'/'on'(case-insensitive) are for True, any string else is for False. Click "update to map" to update the modification to the map immediately.
> 3. "zoom all" to zoom all the map of current activated subwindow.
> 4. "reload" to reload the map of current activated subwindow. This is convenient to view the map after you have finished editing the map xml in your favorite editor.
