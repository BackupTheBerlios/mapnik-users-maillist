Mapnik-users,

As discussed on mapnik-devel [1], we are migrating the central mapnik svn repository to git for hosting on github.

So, svn.mapnik.org as of TODAY is now replaced by git://github.com/mapnik/mapnik.git and can be viewed at https://github.com/mapnik/mapnik

http://svn.mapnik.org will stay available for now but will no longer see development updates.

We will also be migrating the trac wiki to github as time permits over the next several weeks.

If you are interested in helping with the wiki migration please get in touch.

Cheers,

Dane
