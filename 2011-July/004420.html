./bjam is not sufficient to install boost. That may build it, but it does not put in int /usr/local without 'install' as an argument. So I think the problem is simply that you have not *installed* boost.

Try the instructions here:

http://trac.mapnik.org/wiki/Mapnik2#BuildingBoost

Dane
