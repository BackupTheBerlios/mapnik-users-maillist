Magnus,

Mapnik 2.0 (soon to be released trunk) does support setting a scale factor for when you are rendering to png/jpeg output formats.

The reason that ticket exists and that we still need to clean up and document the api (so things may still change in how it is exposed).

But if you want to test the pre-release version then you can do (from python):

scale_factor = 2
mapnik2.render_to_file(map,"map.png","png",scale_factor)

Dane
