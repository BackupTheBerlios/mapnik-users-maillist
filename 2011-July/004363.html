Hi Félix,

Homebrew's ICU formula is currently broken. If you want to make it work you need to rollback to an older version of homebrew itself, see: https://github.com/mxcl/homebrew/issues/6218#issuecomment-1458823

But, yes, the Mapnik 2.0 OSX installer  will work fine with node-mapnik, so I would recommend that route.

Let me know how things work out.

Dane
