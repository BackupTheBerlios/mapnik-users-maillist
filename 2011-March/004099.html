On Mar 18, 2011, at 4:36 AM, Robert Buckley wrote:

> I would like to remove mapnik2 from my system and build mapnik instead, but can not find any information regarding the removal of the application. 
> 
> I tried scons.py uninstall...but this give the error...
> 
> scons: *** No SConstruct file found.
> File "/src/mapnik/scons/scons-local-1.2.0/SCons/Script/Main.py", line 826, in _main

Robert, I think you might be running the command from the wrong directory?

Either way, it is easy to uninstall Mapnik2. Just go into your Mapnik2 source checkout and type:

    sudo python scons/scons.py uninstall

If you are using a globally installed scons then this should work as well:

    sudo scons uninstall
