Hi Mapnik Fans

The first Mapnik Code Sprint: Committers and Cartographers starts next
Friday, the 24th in London and San Francisco.

The code sprint will be a weekend of intense concentration, high drama
and international intrigue.  Hold on.  Wrong announcement.  I'll start
again.

The Mapnik code sprint will be a weekend of removing bugs, adding
features and planning for the future of Mapnik.  If you can't join us
in London or San Francisco, you might think that you can't participate
in the code sprint.  But that isn't true.  We'd love to have you join
us on IRC, and the mapnik-devel@ mailing list to follow along and
participate.  You might think that you can't contribute because your
coding skills are a bit rusty, or because you consider yourself "just
a Mapnik user, not an expert or anything."  But that isn't true
either!  You can contribute too.

Here's how.  Every Mapnik user can contribute by using Mapnik and
filing their bug reports and feature requests.  Even if you aren't
sure that what you are seeing is a bug, do the best you can to file a
bug, perhaps what you are seeing is a the result of some missing
documentation, and we can fix that for you.

Find out more about the Mapnik Code Sprint, file your bugs, and look
forward to great things next weekend.

http://mapnik.org/news/2010/sep/03/msc01/

Best regards,
Richard
